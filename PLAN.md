# MemeOps Plan

## Scope

Ship a static-first `v1` that feels like a small fake-serious product rather than a loose meme dump.

## Phase 1

- establish shared schema conventions
- launch four coherent categories
- generate `docs/api/v1/` from curated source content
- ship a polished static docs and explorer site

## Launch Standard

- 20+ strong entries
- predictable URLs
- consistent object shapes
- reusable JSON for future bots, widgets, and games
- enough product surface area to feel like a real API company with bad incentives

## Next Extensions

- add `postmortems`
- add `severity-translator`
- add `token-guilt`
- add `startup-generator`
- add shareable image cards and themed bundles
